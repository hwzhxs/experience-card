Before adding a new storage layer or subsystem, audit existing layers for overlap and underutilization. This session almost created a knowledge/ directory before discovering that expanding experience-card categories (adding concepts + decisions) achieved the same goal with zero new infrastructure. Complexity is the enemy of accuracy in agent systems.
