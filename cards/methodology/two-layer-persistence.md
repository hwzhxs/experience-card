---
title: Two-layer persistence — real-time save + exit retro
created: 2026-04-09
source: retro from Hive agent analysis
---

Persist knowledge in two layers to prevent session amnesia. Layer 1: save immediately during conversation when corrections, decisions, or insights happen. Layer 2: before session ends ("收工/晚安/wrap up"), scan the conversation for anything not yet saved. Session structure (long vs short) matters less than save discipline — both layers working means any new session can pick up where the last left off.
